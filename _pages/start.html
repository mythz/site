Get Started with ServiceStack

Follow the steps below to explore ServiceStack

Start building your own web services

Explore live demos

Step 1: Install the dotnet x tool

$ dotnet tool install -g x

Step 2: Select a template

Web

x new web WebApp

VueJS

x new vue-spa VueApp

React

x new react-spa ReactApp

Angular

x new angular-spa AngularApp

AWS Lambda

x new aws-lambda AwsLambdaApp
